#TicTacToe
Java Text Game for Terminal
----------------------------------
Program contains two class - TicTacToe and Match

The Program is designed to be Played on Linux terminal using text based commands
---------------------------------
Instructions to Play :-
Select 1 for a 2 Player match; Select 2 for a Player vs CPU match
After that tictac toe grid will appear containing the position no. of each grid.
Enter the desired grid no. [0-8]
The game will continue until either player wins or every grid is filled indicating a draw.
